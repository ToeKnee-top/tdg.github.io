# The Duck Game
* A 2.5D game that can be changed to 3D about a duck who was a hero in his past, but he lost his memory and now has to save the beaten down gods rise back to their former state, fighting obstacles and opponents along the way. (sounds a lot like BOTW, right) This will be a continuation of a simple game that I already started. Will be published on google play, steam, itch, and maybe even nintendo.*
